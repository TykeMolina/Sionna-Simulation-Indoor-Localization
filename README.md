# Avances 2.0

Para poder crear las antenas con dirección es necesario usar la libería de Tensorflow,
esta librería para trabajar en conjunto con Sionna requiere de versiones compatibles de Python,
para evitar varias versiones de python en la computadora, evitando también el uso de Linux (por si 
en un futuro esto se reutiliza y no se quiere usar ese SO), se optó por usar miniconda, que da la
ventaja de crear un ambiente virtual con la versión de Python deseada y versiones diferentes de liberías
y así no mezclarse con el resto de los archivos de la computadora.

Miniconda se puede instalar [Aquí](https://www.anaconda.com/docs/getting-started/miniconda/install/windows-gui-install). 
Yo usé la version 3.10 de Python
